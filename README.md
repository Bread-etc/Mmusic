# Mmusic🎵

一款基于Electron + Vite + React的音乐播放软件,仅做学习用途📦.

## Core Function

- [x] 在线音乐源搜索: 网易云
- [ ] 其他音乐源搜索：如酷狗、QQ、bilibili
- [x] 播放控制（播放/暂停/上一首/下一首/进度条/音量）
- [ ] 音质控制 (standard/higher/extrahight/lossless)
- [x] 扫码登录获取 Cookie
- [ ] 其他平台扫码登录获取 Cookie
- [x] 浅深色模式切换
- [ ] 歌词功能
- [ ] 本地的播放列表（歌单功能）
- [ ] 发现与探索
